**HOW TO RUN THE GAME**
1. Ensure you have Python 3 installed.
2. Open the project using any software that can run Python, such as:
  VS Code
  CodeChum
  Terminal / Command Prompt
  Any Python-capable IDE
3. Run the main game file using:
   python main.py
4. Follow the instructions displayed on screen.

**GAME DESCRIPTION**
Duelmon is a turn-based pet monster battle simulator played entirely on console. Players choose a starter Duelmon and face off against computer-controlled opponents. Each Duelmon has unique moves with varying amounts of damage, and players take turns attacking or defending until a Duelmon is knocked unconscious. The goal is to defeat all opposing Duelmon using strategy and careful move selection. The game features clear win/loss conditions.
